## Descrição do projeto 
Essa aplicação é um sistema de agendamentos de aulas particulares, na qual
o aluno cadastrado no sistema agenda aulas com um professor e tem a possibilidade
de gerir esses agendamentos e seu cadastro no sistema.

## Funcionalidades

Possuímos uma tela de login e cadastro, vinculada a um 
banco de dados SQLite, que permite o usuário se cadastrar e navegar pelo sistema.

o usuário visualizar e gerir seus agendamentos, com as opções de fazer um agendamento,
alterar um agendamento existente e cancelar um agendamento. Além disso, na parte superior
tem os botões que permite o usuário sair do sistema e editar seus dados cadastrados no 
sistema.
